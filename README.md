# HR Analytics Dashboard

## Project Overview

This project focuses on analyzing employee attrition trends using the IBM HR Analytics dataset. The goal was to identify the major factors affecting employee turnover and build an interactive dashboard for business insights.

## Tools Used

* Microsoft Excel
* Power BI
* DAX
* Pivot Tables
* Data Cleaning

## Key Dashboard Features

* Total Employees KPI
* Attrition Count
* Attrition Rate
* Average Salary
* Department-wise Attrition Analysis
* Job Role Analysis
* Age Group Attrition
* Interactive Slicers and Filters

## Key Insights

* Research & Development had the highest employee attrition
* Employees under 31 showed the highest attrition rate
* Lower salary employees were more likely to leave the company

## Business Impact

This dashboard helps HR teams identify high-risk employee groups and improve employee retention strategies.
